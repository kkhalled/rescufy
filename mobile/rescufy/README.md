# Rescufy

Rescufy is a mobile emergency response application that allows users to report emergency cases
and helps reduce ambulance response time through a clear and simple digital flow.

## Current Status
- UI implementation completed
- Screens built with dummy data
- Backend and state logic to be integrated next

## Tech Stack
- Flutter
- flutter_bloc (Cubit + Bloc)
- Clean Architecture (pragmatic)

## Next Steps
- Wire state management
- Integrate backend API
- Add offline support
